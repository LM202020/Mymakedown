## Project Agent Workflow for Codex

Read `AGENTS.md` first.

Before modifying files, confirm:

1. active task
2. owned files
3. files not to touch
4. required verification command
5. known failures
6. next recommended step

Do not modify files outside the owned file list.

After implementation, update:

1. `docs/SESSION_HANDOFF.md`
2. `docs/handoffs/<timestamp>-codex-to-<agent>.md`
3. `docs/HANDOFF_INDEX.md`
4. active plan task status

End with the Agent Output Contract from `docs/AGENT_ROSTER.md`.
